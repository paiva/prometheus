There are 4 java classes:
- Main.java
- XMLGenerator.java
- XMLReader.java
- XMLModifier.java

There is 1 xml file for quick testing:
- ant_2.xml

[[ Notes: ]]

- To create a new XML file, uncomment "writeAntXML.XMLGenerator();" in Main
  and comment "readAntXML.readAntBody();"
- Select the path for the XML file to be created
- Number of body modules created is set to 3 for simplicity and testing, can be modified
- To read XML file, make sure to select the correct path where the XML file was 
created 
- The XMLModifier is suppose to generate the truevalsArray which is unfinished
